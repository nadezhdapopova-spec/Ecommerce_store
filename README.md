# Проект "Ecommerce_store"

## Описание:

"Ecommerce_store" - это учебный проект по Python. 

Проект содержит код для разработки интернет-магазина. 
В проекте используется ООП-подход для создания сервисов и работы с данными.

## Установка:

1. Клонируйте репозиторий проекта:
````
git clone https://github.com/nadezhdapopova-spec/banking-transaction-analyzer.git
````
2. Установите зависимости внутри каталога проекта:
````
poetry install
````

## Использование:

В проекте реализованы классы Product и Category. 

Класс Product создан для хранения информации от товарах: название(name), описание(description), цена(price),
количество в наличии(quantity).

Класс Category помимо атрибутов название(name), описание(description), список товаров категории(products)
позволяет автоматически генерировать информацию о количестве категорий(category_count), общем количестве товаров
(product_count) и количестве товаров в категории.

В проекте реализованы классы Smartphone и LawnGrass, являющиеся подклассами Product. 
Класс Smartphone расширен атрибутами: производительность (efficiency), модель (model), объем встроенной памяти (memory),
цвет (color).
Класс LawnGrass расширен атрибутами: страна-производитель (country), срок прорастания (germination_period), 
цвет (color).
Добавлена функция сложения цены товаров только из одинаковых классов.
Добавление товаров в одну категорию невозможно для товаров из разных классов,
не наследованных от общего базового класса.

Добавлена возможность считывания json-файлов и преобразования информации о товарах и категориях
в экземпляры классов Product и Category.
